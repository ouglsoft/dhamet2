# Dhamet2 Emergency Online

نسخة احتياطية مستقلة للعب ظامت أونلاين فقط على `dhamet2.ouglsoft.com`.

## الصفحات الوظيفية

- `pages/loby.html`: اللوبي والدعوات والغرف والمشاهدة.
- `pages/game.html`: المباراة الأونلاين.

المسار `/` يعيد التوجيه إلى اللوبي. لا توجد صفحة نمط أو دخول أو لوحة تحكم أو لعب ضد الحاسوب.

- زر العودة في اللوبي يفتح `https://ouglsoft.com/dhamet/pages/mode.html`.
- نهاية المباراة تعرض زرًا واحدًا «موافق» ثم تعيد إلى صفحة النمط في التطبيق الأساسي.
- لا توجد إعادة مباراة داخل التطبيق الاحتياطي.
- الهوية Firebase Anonymous فقط، والنتائج غير مصنفة ولا تحفظ في حسابات التطبيق الأساسي.

## الأوامر

```bash
npm test

```
